Reproduzindo um audio

Preload

Indica se o audio sera ou nao pre-carregado e aceita 3 valores:

  • metadata - carrega apenas as informaçoes sobre o arquivo (tamanho, tempo, informaçoes de direitos, etc)
  • none - nao vai carregar nada ate dar play.
  • auto - é o padrão e vai carregar o arquivo inteiro assim que o usuario entar no site, mesmo que nunca de o play.
