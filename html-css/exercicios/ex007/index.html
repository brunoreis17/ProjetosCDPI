Abrindo links em novas guias

o paramentro target define o comportamento do link que o usuario vai abrir. entre os possiveis valores temos:

  • _blank: vai abrir o link em uma nova janela em branco
  • _self: vai abrir na janela atual(link)
  • _top: vai desfazer todos os frames e abrir no navegador completo
  • _parent: similiar ao _top
  • nome-do-frame: caso usamos o frames, indicar o nome da janela ao abrir.

podemos indicar a natureza do destino do link utilizando o parametro rel que aceita os seguintes valores:

  • next: indica que o link é a proxima paete do documento atual.
  • prev:indica que o link é a parte anterior do documento atual.
  • author: indica que é um link para o site do autor do artigo atual.
  • externo: indica um link externo
  • nofollow:indica que é um link para um site nao endossado, como um link pago.
